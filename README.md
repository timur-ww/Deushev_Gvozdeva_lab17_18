# WPFZooManager

## Назначение программы
Простое приложение для управления зоопарками и животными. Программа помогает:
- Вести список зоопарков
- Управлять животными в каждом зоопарке
- Добавлять, изменять и удалять зоопарки и животных

## Функции
- **Добавить зоопарк** - создать новый зоопарк
- **Изменить зоопарк** - отредактировать информацию о зоопарке
- **Удалить зоопарк** - убрать зоопарк из списка

## Как запустить проект

### Шаги для запуска:
1. **Скачать проект с репозитория**
2. **Открыть проект в Visual Studio 2022**
3. **Запустить программу сочетанием клавиш CTRL+F5**
